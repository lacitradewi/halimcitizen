# Halim Citizen
### Halim Citizen is a website that is intended for residents of Halim Perdana Kusuma for search and see data anywhere and anytime. In addition, data can only be seen and taken by the party who has the right to retrieve data. It is made using Adobe Dreamweaver and mySQL.

## New System
### To change data, citizen coming to RT requesting a letter.  After that, the citizen came to RW with a file so the RW Chair can proceeded to replace data via online. RW must log in first. The user id and password are obtained in the Kelurahan. After logging in, RW  checks data citizen in database. If the citizen is not yet registered, RW can enter citizen data into the database. The new citizen's data was then made into a report on the list of new residents. The report was then followed up in Kelurahan. After Kelurahan checks and gives approval to the citizen, data can be updated.
